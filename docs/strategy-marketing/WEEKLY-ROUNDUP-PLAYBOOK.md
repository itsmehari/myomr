## Weekly “This Weekend in OMR” – Playbook

Cadence
- Every Thursday noon: curate events for Sat/Sun; publish roundup
- Friday morning: share to WhatsApp/Telegram using UTM links

Where
- Page: `/local-news/this-weekend-in-omr.php` (auto-curated; you can add intro text on the page if needed)

Steps
1) Review approved events in `/omr-local-events/admin/view-listings.php` for Sat/Sun
2) Ensure titles/locations are clear; feature 1–3 events if needed
3) Publish/update roundup page (it’s dynamic; no manual build needed)
4) Share links from News Highlights CTA and social buttons on the roundup page

UTM policy
- `utm_source=news|events|whatsapp|telegram`, `utm_medium=internal|social`, `utm_campaign=weekend_roundup`

Checklist
- [ ] 5–10 relevant weekend events are visible on roundup
- [ ] News Highlights CTA updated
- [ ] Share to WhatsApp/Telegram with UTM
- [ ] Optional: pin one post on Facebook/Instagram with link
