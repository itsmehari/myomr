## 📋 Executive Summary

This document provides a comprehensive mapping of MyOMR features to UN Sustainable Development Goals (SDGs) and recommends strategic placement of floating SDG badges across the platform to enhance sustainability messaging and community engagement.

## 🎯 Objective

- **Visual Alignment:** Display relevant SDG badges on feature pages to show MyOMR's commitment to sustainability
- **Community Awareness:** Educate users about how each feature contributes to global sustainability goals
- **Consistent Branding:** Reinforce MyOMR's commitment to UN SDGs through visible, contextual badges
- **User Engagement:** Increase awareness and participation in sustainable practices

## 🔗 Feature-to-SDG Mapping

### 1. **Hostels & PGs** (`/omr-hostels-pgs/`)

#### Primary SDG Alignment:

- **SDG 1: No Poverty** ⭐ PRIMARY

  - **Rationale:** Affordable accommodation reduces financial burden, especially for students and young professionals, helping them escape poverty cycles
  - **Impact:** Safe, affordable housing is fundamental to poverty reduction
  - **Badge Priority:** HIGH

- **SDG 11: Sustainable Cities & Communities** ⭐ PRIMARY

  - **Rationale:** Accessible housing in OMR corridor promotes inclusive, safe, and sustainable urban development
  - **Impact:** Reduces urban sprawl, promotes community living, supports local economy
  - **Badge Priority:** HIGH

- **SDG 4: Quality Education** ⭐ SECONDARY

  - **Rationale:** Student-friendly hostels/PGs enable access to quality education by providing affordable accommodation near educational institutions
  - **Impact:** Removes barrier to education (accommodation cost)
  - **Badge Priority:** MEDIUM

- **SDG 8: Decent Work & Economic Growth** ⭐ SECONDARY

  - **Rationale:** Affordable housing for working professionals supports employment opportunities in OMR IT corridor
  - **Impact:** Enables professionals to work in OMR without excessive commute costs
  - **Badge Priority:** MEDIUM

- **SDG 3: Good Health & Well-being** ⭐ TERTIARY
  - **Rationale:** Safe, verified accommodation with amenities (food, security, hygiene) promotes physical and mental well-being
  - **Impact:** Reduces health risks from unsafe housing
  - **Badge Priority:** LOW

**Recommended Badges:**

- **Floating Badge:** SDG 1 (Primary) + SDG 11 (Primary)
- **Position:** Bottom-left corner (opposite WhatsApp button)
- **Display:** Stacked or side-by-side for primary badges

## 🎨 Badge Design Specifications

### Visual Design Requirements:

1. **Size:**

   - **Single Badge:** 60px × 60px (circular or rounded square)
   - **Dual Badges:** 60px × 120px (stacked) OR 120px × 60px (side-by-side)
   - **Mobile:** 50px × 50px (single), 50px × 100px (stacked)

2. **Colors:**

   - Use official UN SDG colors (from existing SDG page CSS)
   - SDG 1: #E5243B (Red)
   - SDG 2: #DDA63A (Yellow)
   - SDG 3: #4C9F38 (Green)
   - SDG 4: #C5192D (Red)
   - SDG 8: #A21942 (Magenta)
   - SDG 9: #FD6925 (Orange)
   - SDG 11: #FD9D24 (Orange)
   - SDG 15: #56C02B (Green)
   - SDG 16: #00689D (Blue)
   - SDG 17: #19486A (Dark Blue)

3. **Icon/Number:**

   - Display SDG number prominently (1-17)
   - Optional: Small icon representing the goal
   - Font: Bold, white color, readable at small size

4. **Position:**

   - **Default:** Bottom-left corner (opposite WhatsApp button)
   - **Z-index:** 99 (below WhatsApp button at z-index 100)
   - **Offset:** 40px from bottom, 40px from left

5. **Animation:**

   - Subtle hover effect (scale 1.1)
   - Optional: Fade-in on page load
   - Smooth transitions (0.3s ease)

6. **Interaction:**
   - Click opens tooltip or links to SDG explanation
   - Tooltip shows: SDG name, brief description, "Learn More" link to SDG page

## 🔧 Technical Implementation

### File Structure:

```
/components/
  └── sdg-badge.php              # Main badge component
/assets/
  └── css/
      └── sdg-badge.css          # Badge styles
/assets/
  └── js/
      └── sdg-badge.js           # Badge interaction logic
```

### Component API:

```php
// Basic usage
<?php include 'components/sdg-badge.php'; ?>

// With custom SDGs
<?php
  $sdg_badges = [1, 11]; // Array of SDG numbers
  include 'components/sdg-badge.php';
?>

// With custom position
<?php
  $sdg_badges = [8, 9];
  $sdg_position = 'bottom-right'; // bottom-left, bottom-right, bottom-center
  include 'components/sdg-badge.php';
?>
```

### Configuration Options:

- **Auto-detect:** Badge component automatically detects page type and displays relevant SDGs
- **Manual override:** Allow manual specification of SDGs via PHP variables
- **Responsive:** Automatically adjusts size and position on mobile
- **Accessibility:** ARIA labels, keyboard navigation support
