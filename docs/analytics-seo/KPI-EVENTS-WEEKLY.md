## Weekly KPI Sheet – Events (Template)

Columns
- Week starting (date)
- Events submitted
- Events approved
- Unique sessions to /omr-local-events/
- CTR: listing → detail (%)
- Shares (sum of channels)
- Ticket clicks
- Newsletter signups (events pages)
- Roundup page sessions

How to track
- Pull from GA dashboard (see `docs/GA-DASHBOARD-EVENTS.md`)
- Submissions/approvals from admin stats

Notes
