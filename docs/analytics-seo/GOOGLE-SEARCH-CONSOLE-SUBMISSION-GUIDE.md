## 📋 Pre-Submission Checklist

### Before Submitting
- [ ] All landing pages are live and accessible

---

## 🚀 Step 1: Verify Website in Google Search Console

### If Already Verified
- ✅ Skip to Step 2

### If Not Verified
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add Property"
3. Enter property type: **URL prefix**
4. Enter: `https://myomr.in`
5. Choose verification method:
   - **HTML file upload** (recommended)
   - **HTML tag** (add to homepage)
   - **DNS record** (if you have access)
   - **Google Analytics** (if GA is installed)
6. Follow verification instructions
7. Wait for verification (usually instant)

---

## 🗺️ Step 2: Submit Sitemap

### 2.1 Update Sitemap

The sitemap has been updated to include all landing pages. Verify it includes:

**Main Sitemap:** `https://myomr.in/sitemap.xml`
- Includes all landing pages
- Includes main job portal pages

**Job Portal Sitemap:** `https://myomr.in/omr-local-job-listings/sitemap.xml`
- Includes all landing pages
- Includes job detail pages

### 2.2 Submit Sitemap in Google Search Console

1. Go to Google Search Console
2. Select your property (`https://myomr.in`)
3. Click **Sitemaps** in left sidebar
4. In "Add a new sitemap" section, enter:
   - `sitemap.xml` (main sitemap)
   - `omr-local-job-listings/sitemap.xml` (job portal sitemap)
5. Click **Submit**
6. Wait for processing (usually 1-24 hours)

## 🔍 Step 4: Monitor Indexing Status

### Check Coverage Report
1. Go to **Coverage** in Search Console
2. Check for:
   - **Valid** pages (indexed)
   - **Excluded** pages (not indexed)
   - **Errors** (fix any issues)

### Check URL Inspection
- Use URL Inspection tool to check individual pages
- Verify pages are indexed
- Check for any indexing errors

---

## 📊 Step 5: Monitor Performance

### Performance Report
1. Go to **Performance** in Search Console
2. Monitor:
   - **Impressions** (how often pages appear in search)
   - **Clicks** (how often users click)
   - **CTR** (click-through rate)
   - **Average Position** (ranking position)

### Filter by Landing Pages
- Filter by landing page URLs
- Track performance over time
- Identify top-performing pages
